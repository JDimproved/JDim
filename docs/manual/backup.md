<document header="バックアップ、アンインストールについて">


<group header="バックアップ方法">
  <sentence>
  キャッシュのバックアップはキャッシュディレクトリ(デフォルトでは ~/.jd )
  以下を保存するだけでよい。
  </sentence>
</group>


<group header="アンインストール方法( rpm の場合 )">
<shell>
$ rpm -e jd
$ rm -rf &lt;キャッシュディレクトリ&gt;
</shell>
</group>


<group header="アンインストール方法( 手動の場合 )">
<shell>
$ rm (インストールパス)/jd
$ rm -rf &lt;キャッシュディレクトリ&gt;
</shell>
</group>


</document>
