### 移転処理
「`ファイル`」メニューから板リストを再読込したときに移転した板があるときは自動で移転処理する。
なお移転処理をしたあとは一度JDimを再起動することを推奨する。

移転情報テーブルはキャッシュディレクトリ内の「`move.info`」ファイルに記録されるので
もしテーブルが壊れたときは手動で修正すること。

#### move.infoの形式
```
旧サーバー 新サーバー /板ID
```

#### 実際の例
```
http://pc2.2ch.net http://pc8.2ch.net /linux
```
